All logs

Get all rental logs from the database.

method: GET

endpoint: /api/v1/rentallog

JSON body the API expects:

                
                    {}

Example response:

                
                    [
                        {
                            "id": 1,"
                            car_id": 1,
                            "user_id": 1,
                            "rented_on": "2020-12-01",
                            "returned_on": "2020-12-03"
                        }
                    ]
