All cars

Get all cars from the database.

method: GET

endpoint: /api/v1/cars

JSON body the API expects:

                
                    {}

Example response:

                
                    [
                        {
                            "id": 1,
                            "make": "Lada",
                            "model": "1200",

Add car

Add a car to the database.

method: POST

endpoint: /api/v1/cars/add

JSON body the API expects:

                
                    {
                        "make":"Toyota",
                        "model": "Corolla",
                        "year": 2015,
                        "color": "white",
                        "license_plate": "ABC123"
                    }

Example response:

                
                    {
                        "status": 200,
                        "message: "Car added successfully"    
                    }

Remove a car

Remove a car from the database.

method: DELETE

endpoint: /api/v1/cars/remove

JSON body the API expects:

                
                    {
                        "id": 1
                    }

Example response:

                
                    {
                        "status": 200,
                        "message: "Car added successfully"
                    }

Rent a car

Rent a car from the database.

method: PATCH

endpoint: /api/v1/cars/rent/CARID

JSON body the API expects:

                
                    {
                        "renter_id": 1,
                        from_date: "2015-01-01",
                        to_date: "2015-01-02"
                    }

Example response:

                
                    {
                        "status": 200,
                        "message: "Car rented successfully"
                    }
