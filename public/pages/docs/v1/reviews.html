All reviews

Get all reviews from the database.

method: GET

endpoint: /api/v1/reviews

JSON body the API expects:

                
                    {}

Example response:

                
                    [
                        {
                            "id": 1,
                            "car_id": 1,
                            "user_id": 1,
                            "rating": 5,
                            "review": "Great car!",
                            "created_at": "2024-12-01"
                        },
                        {
                            "id": 2,
                            "car_id": 3,
                            "user_id": 6,
                            "rating": 3,
                            "review": "The car was dirty when I got it",
                            "created_at": "2023-05-06"
                        }
                    ]

Add review

Add a review to the database.

method: POST

endpoint: /api/v1/reviews/add

JSON body the API expects:

                
                    {
                        "car_id": 1,
                        "user_id": 1,
                        "rating": 5,
                        "review": "Great car!"
                    }

Example response:

                
                    {
                        "status":200,
                        "message": "Review added."
                    }

Remove a review

Remove a review from the database.

method: DELETE

endpoint: /api/v1/reviews/remove

JSON body the API expects:

                
                    {
                        "id": 1
                    }

Example response:

                
                    {
                        "status": 200,
                        "message": "Review removed."
                    }
