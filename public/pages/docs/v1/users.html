All users

Get all users from the database.

method: GET

endpoint: /api/v1/users

JSON body the API expects:

                
                    {}

Example response:

                
                    [
                        {
                            "id": 1,
                            "email": "test@example.com",
                            "email_verified": 0,
                            "first_name": "John",
                            "last_name": "Doe",
                            "admin": 0,
                            "times_rented": 8,
                            "currently_renting": 2
                        },
                        {
                            "id": 2,
                            "email": "test2@example.com",
                            "email_verified": 1,
                            "first_name": "Jane",
                            "last_name": "Doe",
                            "admin": 0,

                    ]

Add user

Add a user to the database.

method: POST

endpoint: /api/v1/users/add

fname = First name
lname = Last name
email = Email
password = Password

JSON body the API expects:

                
                    {
                        "fname": "John",
                        "lname": "Doe",
                        "email": "test@domain.com",
                        "password": "passwordHere"
                    }

Response:

                
                    {
                        "status":200, 
                        "message":"User created successfully"
                    }

Login

Login to the system.

method: POST

endpoint: /api/v1/users/login

email = Email
password = Password

JSON body the API expects:

                
                    {
                        "email": "test@domain.com",
                        "password": "passwordHere"
                    }

Response:

If login is successful, it will return a JSON object with a token that you can use to authenticate your requests.

                
                    { 
                        "status": 200, 
                        "message": "Logged in successfully", 
                        "token": "ACCESS_TOKEN", 
                        "admin": 0 
                    }

Update user information

Update user information.

method: PUT

endpoint: /api/v1/users/update

fname = First name
lname = Last name
email = Email
password = Password

JSON body the API expects:

                
                    {
                        "fname": "John",
                        "lname": "Doe",
                        "email": "john.doe@example.com",
                        "admin": 1
                    }

Response:

                
                    {
                        "status":200, 
                        "message":"User updated successfully"
                    }
